## Serrano Server

### Build instructions:

    npm install
    grunt

### To start the server:

    cd build
    node server.js

### To use the server:

    HTTP GET http://<your-ip>:3000/doc/<doc-id>/version
    HTTP GET http://<your-ip>:3000/doc/<doc-id>/data
